|------|------|--------|
| 첫 로그인만 실패, 재시도는 성공 | DB 콜드스타트가 OAuth 콜백 타임아웃 유발 / 게스트 세션 잔존 | 로그인 직전 /api/health 응답시간 확인 |

| 산발적 "시스템 오류" (여러 라우트 동시) | DB 커넥션 풀 고갈 (세션 모드 15개 한도) | Vercel 로그에서 EMAXCONNSESSION 검색 |

| 글자마다 TTS 목소리 남/여 뒤바뀜 | 레이트리밋 초과분이 브라우저 내장 음성 폴백 | 연속 재생 20+회 후 목소리 확인 |
| 콘텐츠 하단 배경 잘림/버튼 가림 | min-h-dvh가 부모 padding 밖 / 고정 네비와 겹침 | 짧은 콘텐츠 페이지에서 스크롤 최하단 확인 |
| 스쿼시 머지했는데 main에 변경 없음 | 같은 브랜치에서 추가+삭제 → 순변화 0 | merge-base 기준 diff 확인 |
| 머지 후 프로덕션 미반영 | 프로덕션 배포 유실/큐 정체 (프리뷰만 성공) | 새 엔드포인트/문자열로 프로덕션 폴링 |
