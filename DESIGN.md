# SORI 디자인 핸드오프 브리핑

## 프로젝트 개요

**앱명:** 소리새 (SORI) — 아동 조음 홈케어 SaaS  
**성격:** 아이와 부모가 함께 발음 교정 연습을 하는 게임화된 앱  
**레이아웃:** 모바일 전용 (`max-w-lg`, 512px), 가운데 정렬

## 디자인 시스템 — "마시멜로 파스텔"

### 컬러 팔레트

| 역할 | 이름 | Hex |
|------|------|-----|
| 배경 | 아이보리 | `#FDFAF5` |
| 배경(서브) | 피치 틴트 | `#FFF5EE` |
| 주 액션 | 피치 | `#FFB38A` |
| 피치 라이트 | — | `#FFD4B8` |
| 성공 / 보조 | 민트 | `#7EDFD0` |
| 민트 라이트 | — | `#B8EDE3` |
| 정보 / 팁 | 라벤더 | `#C4B5FD` |
| 라벤더 라이트 | — | `#EDE9FE` |
| 강조 | 옐로우 | `#FDE68A` |
| 오류 / 교정 필요 | 핑크 | `#FCA5A5` |
| 주 텍스트 | 웜 다크 | `#3D3530` |
| 보조 텍스트 | 웜 그레이 | `#8B7E74` |
| 테두리 | 마시멜로 | `#F0E8E0` |

### 둥근 모서리 (마시멜로 테마)

- `sm`: 12px
- `md`: 20px
- `lg`: 28px
- `xl`: 36px
- `full`: 9999px

### 그림자

- **bubble:** `0 4px 20px rgba(180,120,80,0.10)`, `0 1px 4px rgba(180,120,80,0.08)`
- **hover:** `0 8px 32px rgba(180,120,80,0.15)`, `0 2px 8px rgba(180,120,80,0.10)`
- **card:** `0 2px 12px rgba(180,120,80,0.08)`

### 타이포그래피

- **폰트:** Pretendard Variable
- **폴백:** Pretendard, -apple-system, BlinkMacSystemFont, "Apple SD Gothic Neo", "Noto Sans KR", sans-serif
- 제목: `font-black` (900)
- 섹션헤더: `font-bold` (700)
- 라벨: `font-semibold` (600)
- 글자 크기: 36px(히어로) → 24px → 20px → 16px → 14px → 12px

---

### 1. 랜딩 페이지 `/`

**배경:** 3색 그라디언트 `피치 → 민트 → 라벤더 (160deg)`

| 구역 | 내용 |
|------|------|
| 히어로 | 소리새 마스코트 SVG (파랑새, 150px) + float 애니메이션, 앱 슬로건, 피치 CTA 버튼 |
| 피처 카드 | 3열 그리드, 흰 BubbleCard, 이모지 + 설명문구 |
| 프라이싱 CTA | 단일 카드, 민트 or 라벤더 배경, 구독 버튼 |

---

### 2. 로그인 `/login`

**배경:** 그라디언트 동일

- 중앙 흰 BubbleCard 1개
- `🐣` 이모지 float 애니메이션
- 카카오 버튼 (`#FEE500` 노랑 배경, 검은 텍스트)
- 구글 버튼 (흰 배경, SVG 아이콘, 라이트 보더)
- 두 버튼 모두 full-width, `border-radius: 9999px`

---

### 3. 온보딩 `/onboarding`

- BubbleCard 폼 1개
- 아이 이름 입력 (필수) — 피치 포커스 보더
- 생년월일 입력 (선택)
- 피치 버튼으로 제출

---

### 4. 대시보드 홈 `/dashboard`

**헤더:** 고정(sticky), 글래스모피즘 (`rgba(253,250,245,0.85)` + blur 12px)  
**하단 내비:** 5개 탭 (홈 / 연습 / 기록 / 즐겨찾기 / 아이모드), 동일 글래스 처리

| 섹션 | 내용 |
|------|------|
| 인삿말 | 소리새 로고(56px) + 레벨 배지(피치, 절대위치) + 이름 + 🔥 연속 뱃지 |
| 통계 3열 | "학습 단어수 / 연습시간 / 연속일수" — 각 카드에 이모지 아이콘 |
| 오늘의 미션 | 피치 보더(진행중) / 민트 보더(완료), 음소 배지들, 완료 토글 버튼 |
| 주간 비교 | 이번 주 vs 지난 주 점수 bar |
| 최근 세션 | 날짜 + 단어수 + 정확도 리스트 |
| 형제 비교 | 피치 바 (주 아이) vs 민트 바 (형제), 2명 이상일 때만 표시 |
| 아이모드 CTA | 큰 피치 버튼 |

### 5. 세션 시작 `/dashboard/session/new`

| 구역 | 내용 |

| 아이 선택 | 상단 드롭다운 or 탭 |

| 단어 선택 | 3열 그리드 — 이모지 + 단어, 선택 시 피치/민트 bg + 진한 보더 |
| 시작 버튼 | "N개 단어 연습 시작" — 하단 고정 피치 버튼 |

### 6. 세션 연습 `/dashboard/session/[id]`

**가장 핵심 화면 — 연습 플로우**

| 요소 | 상세 |
|------|------|
| 상단 진행바 | 피치 fill, `현재/전체` 표시 |
| 목표 단어 카드 | 피치 bg, 이모지(크게), 단어(font-black 크게), 음절 분리(· 구분자), 의성어 힌트(italic), 예문 |
| 시도 도트 5개 | 회색→피치(틀림)→민트(맞음), 각 채워질 때 pop 애니메이션 |
| 음절 안내 (mint) | 첫 시도 시에만 표시, 민트 bg, 음절별 발음 안내 |
| 부모 입력 폼 | "아이가 어떻게 발음했나요?" 텍스트필드, 피치 포커스, 분석하기 버튼 |
| 분석 결과 카드 | 오류 음소 표시 (`ㅅ→ㄷ`), 핑크 뱃지 |
| 단어 완료 배너 | 정답: 민트 bg "잘했어요!", 연습 필요: 흰 배경 |
| 완료 화면 | 점수 + 마스코트 + 피치 버튼 |

### 7. 아이 모드 `/dashboard/child`

**풀스크린 게임화 모드 — 아이가 직접 사용**

- 배경: `135deg 피치→민트→라벤더` 그라디언트 전체화면
- 상단 진행 도트 (피치=완료, 반투명 흰=대기)
- 마스코트 캐릭터 (레벨별 글로우 링)
- 단어 카드: 이모지 (`text-8xl`), 단어 (`text-5xl font-black`), `border-radius: 32px`
- 하단 2버튼: "맞아요! 🌟" (피치, 크게) / "다음" (흰 배경)
- 정답 시 confetti 폭발 (피치/민트/라벤더/노랑/핑크 파티클, 별+원 모양)

## 인터랙션 상태

### 버튼
- **기본:** shadow-bubble
- **Hover:** translateY -1px, shadow-hover
- **Active:** scale 0.96
- **Disabled:** opacity 50%, cursor not-allowed

### 입력 필드
- **기본 보더:** 2px `#F0E8E0`
- **포커스 보더:** 2px `#FFB38A` (피치)
- **플레이스홀더:** `#C4B5A8`

### 카드 선택 (단어/모드 선택)
- **미선택:** 흰 bg, 라이트 보더
- **선택:** 피치/민트 bg, 진한 컬러 보더 2px

## STITCH / Figma 전달 핵심 메모

1. **모바일 단일 레이아웃** — 데스크탑 버전 없음, 최대폭 512px, 세로 스크롤
2. **한국어 전용** — 모든 카피는 한국어, Pretendard 폰트 필수
3. **마시멜로 테마** — 직각/날카로운 모서리 전혀 없음, 최소 radius 12px
4. **색상 규칙:** 피치=주 액션, 민트=성공/보조, 핑크=오류 — 이 3색 규칙 엄수
5. **글래스모피즘** — 헤더/하단 nav만 적용 (blur + 반투명 배경)
6. **애니메이션 필수 구현:** float, bounce-in, dot-pop, confetti
7. **마스코트 소리새** — 파랑새 SVG 캐릭터, 자체 blink/wing 애니메이션 내장
8. **레벨 시스템** — 5단계 (아기 소리새 → 전설의 소리새), 각 레벨별 글로우 색상 변화
