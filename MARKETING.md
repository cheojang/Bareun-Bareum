# 바른발음 런칭 마케팅 계획서

## 1. 타깃 고객 정의

| 세그먼트 | 설명 | 우선순위 |
|----------|------|----------|
| **A. 치료 병행 부모** | 언어치료 센터에 다니는 아이를 둔 부모 — 집에서도 연습하고 싶은 니즈 명확 | ★★★ |
| **B. 관찰 중 부모** | "우리 아이 발음이 좀 이상한 것 같은데" 단계 — 치료까지는 아니지만 걱정되는 부모 | ★★★ |
| **C. 예비 치료 부모** | 대기 중이거나 비용 부담으로 치료소 방문을 미루고 있는 부모 | ★★ |

**핵심 페르소나:** 5세 아이를 둔 30대 초반 엄마, 맘카페 활동 중, 스마트폰으로 육아 정보 수집

> **"언어치료사 없이도, 집에서 5분으로 아이 발음 교정"**

- "월 5,000원으로 전문가 처방전"

## 3. 채널별 홍보 전략

### 🟢 1순위 — 네이버 (검색 의도 기반)

**블로그 SEO 콘텐츠**
- 키워드: `아이 발음 교정`, `5살 발음 이상`, `ㅅ발음 안되는 아이`, `언어치료 비용`
- 월 4~8편 포스팅 (직접 운영 또는 체험단 활용)
- 글 형식: "○○ 발음이 안 되는 이유 + 집에서 할 수 있는 방법"

**네이버 카페 (맘카페)**
- 타깃 카페: 맘카페 (회원 수 상위), 지역 육아 카페, 언어치료 관련 카페
- 전략: 광고 티 나지 않는 후기 형식 — "우리 아이 ㅅ발음 교정 3주 후기"
- 주의: 카페 규정 확인 후 자연스러운 체험 후기로 접근

**네이버 지식iN**
- "아이 발음 이상한데 어떻게 해요?" 류 질문에 성실한 답변 + 자연스러운 언급

---

### 🟠 2순위 — 인스타그램 / 유튜브 쇼츠

**인스타그램**
- 계정 테마: 육아 발음 교정 팁 + 앱 사용 스토리
- 콘텐츠 형식:
  - 카드뉴스: "ㄹ발음 안 되는 이유 5가지"
  - 릴스: 앱으로 발음 분석하는 30초 영상
  - 스토리 투표: "우리 아이도 이런 발음 해요?"
- 해시태그: `#육아`, `#발음교정`, `#언어발달`, `#5세육아`, `#아이발음`

**유튜브 쇼츠 / 틱톡**
- "이 발음 못 하면 언어치료 필요할 수 있어요" 시리즈
- 발음별 원인 설명 30~60초 클립
- 앱 데모 영상 (입력 → 분석 → 훈련법 확인 흐름)

---

### 🔵 3순위 — 카카오 / 커뮤니티

**카카오톡 채널**
- 채널 개설 → 구독자 대상 주 1회 발음 팁 + 이벤트 알림
- 가입 유도: 앱 회원가입 시 카카오 채널 구독 → 무료 체험 연장 쿠폰

**맘카페 / 오픈카톡방**
- 언어치료 관련 오픈카톡방 참여 → 자연스러운 정보 공유
- "발음 걱정 부모 모임" 오픈채팅방 직접 운영

---

### 🟣 4순위 — 오프라인 / B2C 협력

- **언어치료 센터 협력** (2단계 예정이지만 마케팅 채널로도 활용 가능)
  - 대기실 QR 코드 스티커: "집에서도 연습하세요"
  - 치료사가 부모에게 추천하는 홈케어 앱으로 포지셔닝
- **어린이집 / 유치원**
  - 원장 대상 안내문 배포 또는 이메일 발송
  - "발음 걱정 아이 부모님께 드리는 앱" 홍보물

## 4. 런칭 이벤트 & 프로모션

### 🎉 런칭 D-Day: 얼리버드 캠페인 (출시 후 2주)

| 항목 | 내용 |
|------|------|
| **혜택** | 프리미엄 첫 달 무료 (기존 첫 달 무료와 별도로 한 달 추가) |
| **조건** | 회원가입 후 발음 분석 1회 이상 완료 |
| **목표** | 초기 사용자 데이터 확보 + 리뷰 수집 |

---

### 📸 후기 이벤트 (출시 후 1개월)

| 항목 | 내용 |
|------|------|
| **미션** | 앱 사용 후기를 블로그/인스타에 올리고 URL 제출 |
| **상품** | 추첨 10명 — 스타벅스 아메리카노 5잔 기프티콘 |
| **효과** | 자연 검색 유입 콘텐츠 생성, 신뢰도 상승 |

---

### 👨‍👩‍👧 친구 초대 이벤트 (상시)

| 항목 | 내용 |
|------|------|
| **구조** | 초대 링크로 가입한 친구가 유료 전환 시 → 나도 1개월 무료 연장 |
| **포인트** | 맘카페 특성상 "친구한테 공유" 행동이 자연스러움 |

---

### 🏆 발음 챌린지 (출시 후 2개월, SNS 바이럴용)

| 항목 | 내용 |
|------|------|
| **형식** | 아이가 특정 단어 발음하는 영상 + "바른발음으로 교정 중" 해시태그 |
| **상품** | 매달 우수 후기 3팀 — 아동 도서 세트 |
| **효과** | UGC(사용자 생성 콘텐츠) 확보, 바이럴 확산 |

---

### 🤝 언어치료사 인플루언서 협력 (출시 후 3개월)

- 인스타그램 팔로워 1~5만 언어치료사 계정 5~10개 섭외
- 협력 형태: 무상 프리미엄 제공 + 솔직 후기 요청 (뒷광고 표기 필수)
- 치료사 신뢰도를 빌려오는 효과 — 타깃 부모층에 직접 접근 가능

## 5. 월별 실행 로드맵

```
Month 1 (출시)
├── 얼리버드 캠페인 시작
├── 네이버 블로그 SEO 포스팅 시작 (주 2회)
├── 인스타그램 계정 개설 및 콘텐츠 10개 예약 업로드
└── 카카오 채널 개설

Month 2
├── 후기 이벤트 실시
├── 맘카페 체험 후기 바이럴 (5~10개 카페)
├── 유튜브 쇼츠 5편 업로드
└── 친구 초대 기능 앱 내 적용

Month 3
├── 발음 챌린지 캠페인
├── 언어치료사 인플루언서 1~3명 협력 시작
└── 데이터 분석 — 채널별 전환율 파악 후 예산 재배분
```

## 6. 예산 가이드 (초기 소규모 런칭 기준)

| 항목 | 예상 비용 |
|------|-----------|
| 블로그 체험단 (5명) | 50만원 |
| 인스타 인플루언서 (마이크로 2~3명) | 30~50만원 |
| 이벤트 상품 (기프티콘 등) | 20만원 |
| 네이버 광고 (CPC, 초기 테스트) | 30만원 |
| **합계** | **약 130~150만원** |

> 초기엔 유료 광고보다 **맘카페 바이럴 + SEO**에 집중하는 것이 ROI가 높습니다.
