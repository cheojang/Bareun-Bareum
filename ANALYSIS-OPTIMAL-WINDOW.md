```
200개: 200 × 100B = 20KB

- 100개: 10MB
- 200개: 20MB
- 300개: 30MB
- 500개: 50MB
- 1000개: 100MB
```

**평가:** 모두 무시할 수준. DB 용량은 고려 대상 아님.

### 3. 정확도 & 통계 신뢰도

#### 통계학적 표본 크기

```
신뢰 수준 95%, 오차율 5% 기준으로
필요한 최소 표본 크기 계산:

n = (Z² × p × (1-p)) / E²
- Z = 1.96 (95% 신뢰도)
- p = 0.5 (최악의 경우)
- E = 0.05 (5% 오차율)

n = (1.96² × 0.5 × 0.5) / 0.05²
  = 384 샘플

따라서 최소 384개가 필요함
```

#### 실제 응용 (아동 발음 오류율)

```
100개:  ±9.8% 오차율   (낮음, 신뢰도 떨어짐)
예시: ㅅ 오류율 45% → 실제는 35~55% 사이

200개:  ±6.9% 오차율   (중간 수준)
예시: ㅅ 오류율 45% → 실제는 38~52% 사이

300개:  ±5.7% 오차율   (좋음)
예시: ㅅ 오류율 45% → 실제는 39~51% 사이

400개:  ±4.9% 오차율   (매우 좋음)
예시: ㅅ 오류율 45% → 실제는 40~50% 사이

500개:  ±4.4% 오차율   (우수함)
예시: ㅅ 오류율 45% → 실제는 40.6~49.4% 사이
```

**평가:**
- 100개: 너무 작음. "일시적 실수" 영향 크음
- 200개: 괜찮지만, 여전히 변동성 큼
- 300~400개: **이상적 범위** (5~7% 오차)
- 500개 이상: 과도함 (수익성 vs 정확도 트레이드오프)

### 4. 시간 범위 분석

```
평균 주 3회 연습 기준:

100개:  약 7주 (약 2개월)
200개:  약 13주 (약 3개월)
300개:  약 20주 (약 5개월)
400개:  약 27주 (약 6개월)
500개:  약 33주 (약 8개월)
```

**평가:**
- 100개 (2개월): 너무 짧음. 계절 변화, 휴식 기간 영향 받음
- 200개 (3개월): 한 학기. 나쁘지 않음
- 300~400개 (5~6개월): **반년간 데이터** = 가장 현실적
- 500개 (8개월): 너무 김. 과거 패턴까지 포함

---

### 5. 약점 판정 신뢰도

#### 시나리오: ㅅ 음소 약점 판정

```
실제 약점인지 확인하려면?
→ 최소 30번 이상 시도 후 30% 이상 오류율

100개 데이터:
- ㅅ 포함 단어 약 15~20개 → 오류율 신뢰도 낮음
- 한두 번 실수로도 50% 오류율 나옴

200개 데이터:
- ㅅ 포함 단어 약 30~40개 → 오류율 신뢰도 중간
- 실제 약점 vs 일시 실수 구분 가능

300개 데이터:
- ㅅ 포함 단어 약 45~60개 → 오류율 신뢰도 높음
- 약점의 정도(경미/중간/심각)까지 구분 가능

500개 데이터:
- ㅅ 포함 단어 약 75~100개 → 오류율 신뢰도 매우 높음
- 과도함. 변화 감지가 더뎌짐
```

**평가:**
- 300개: 각 음소별 30회 이상 데이터 보장 (통계적 충분성)
- 400개: 각 음소별 40회 이상 (최적)

## 부모 경험 관점

### 약점 변화 추적

```
예시: ㅅ 음소 개선 추이

100개 기준 (2개월):
- "최근 2주: 70% 오류"
- 변화가 너무 빠르게 나타남 (신뢰도 낮음)

300개 기준 (5개월):
- "최근 2주: 70%" → "지난주: 65%" → "이번주: 60%"
- 확실한 개선 추세 보임 (부모 동기부여 ↑)

500개 기준 (8개월):
- 장기 추세는 명확하지만, 단기 변화 감지 어려움
- "이 정도면 언제 개선되나?" 느낌
```

**평가:** 300개가 부모에게 가장 긍정적인 피드백 제공

### 나이별 데이터 축적 능력

```
만 3세 아이 (발음 아직 발달 중):
- 월 10회 연습 → 300개 데이터 = 30개월 (2.5년)
- "너무 오래 걸림" → 100개가 나음

만 4~5세 아이 (집중 교정기):
- 월 15회 연습 → 300개 데이터 = 20개월 (약 1년 8개월)
- 적당함

만 6세 이상 (심각한 발음 문제):
- 월 20회 연습 → 300개 데이터 = 15개월 (약 1년 3개월)
- 적당함
```

**평가:** 연령별로 데이터 축적 속도가 다르니, "시간 기반" 정책도 필요
